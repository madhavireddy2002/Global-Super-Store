# Global-Super-Store
Data analyst projects

# Project Title
Global Superstore Sales Analysis using Python

# Project Description
This project is completed as part of the CodeAlpha Data Analytics Internship. The objective of this project is to perform Exploratory Data Analysis (EDA) on the Global Superstore dataset and extract meaningful business insights using Python.

The project focuses on understanding sales performance, profit trends, customer behavior, and product categories through statistical analysis and data visualization.

# Dataset
Dataset Used: Global Superstore Dataset

The dataset contains information about:

Order Date
Customer Name
Region
Country
Category
Sub-Category
Product Name
Sales
Profit
Quantity
Discount
Shipping Mode

# Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

# Steps Performed
# Imported required libraries.
Loaded the Global Superstore dataset.
Checked dataset structure using info() and describe().
Identified missing values and duplicates.
Performed data cleaning.
Conducted exploratory data analysis.
Created visualizations for better understanding.
Derived business insights.
# Visualizations
Sales by Region

# Key Insights
Some regions contribute significantly higher sales.
Technology category generates higher profits.
Customer segments show different purchasing patterns.
Discounts affect profitability.
Sales trends vary over time.
